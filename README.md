# Noniee

**Comfort food to remember.**

A website for Noniee — healthy, earthy catering with the warmth of nonna's kitchen.

## Design

- **Palette:** cream, terracotta, olive green, warm brown
- **Typography:** Cormorant Garamond (headings) + DM Sans (body)
- **Vibe:** rustic Italian nonna, seasonal, wholesome, slow-cooked

## Sections

1. **Hero** — Brand introduction & CTAs
2. **Our Story** — Nonna-inspired origin
3. **Philosophy** — Seasonal, whole, slow
4. **Catering** — Intimate gatherings, corporate, special occasions
5. **Menu** — Dish highlights
6. **Contact** — Inquiry form

## Run locally

Open `index.html` in your browser, or serve with any static server:

```bash
# Python
python3 -m http.server 8080

# Or with npx (if Node is installed)
npx serve .
```

Then visit [http://localhost:8080](http://localhost:8080).

## Customize

- Update contact details in `index.html` (email, phone, Instagram)
- Replace Unsplash placeholder images with your own photography
- Adjust menu items and catering packages to match your offerings
- Connect the contact form to a backend or service (Formspree, Netlify Forms, etc.)

## Deploy

This is a static site — deploy anywhere:

- [Netlify](https://netlify.com) — drag & drop the folder
- [Vercel](https://vercel.com) — connect the repo
- [GitHub Pages](https://pages.github.com) — enable in repo settings
